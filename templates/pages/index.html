About Freeminer

Freeminer is a Minecraft-inspired sandbox game.

Freeminer is based on Minetest/Luanti which is developed by a number of contributors from all over the globe.

Unlike Minetest/Luanti it aims to make the game fun while trading off some bits of perfectionism.
